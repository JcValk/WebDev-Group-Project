# WebDev-Group-Project
this is a school project, with 4 members. Our task is create secure, responsive, database-driven web application with clear delivery milestones, documentation, and a deployable build.
This project involves the development of a secure, responsive, and database-driven web application created collaboratively by a team. It demonstrates practical skills in frontend and backend development, database integration, and the implementation of basic security principles.

The system supports user interaction through features such as authentication and CRUD functionality while ensuring responsiveness across different devices. Security and data protection are built into the design through measures like input validation and controlled access to stored information.

The project is divided among four team members with structured milestones to ensure efficient development and timely delivery of a functional final product.

Artificial intelligence tools were used to assist in refining and improving the clarity of written content. All design decisions, development work, and final implementation were completed by the project team.
